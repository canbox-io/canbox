# Launcher 增强功能规划

## Phase 2：增强功能

### 1. 计算器功能

**描述**：在搜索框输入数学表达式（如 `1+2*3`），直接显示计算结果。

**实现方案**：
- 使用 `eval()` 或 `math.js` 库
- 输入时实时计算（防抖 300ms）
- 按 `Enter` 复制结果到剪贴板

**优先级**：⭐⭐⭐

---

### 2. 网页搜索

**描述**：选中文字后，按全局快捷键弹出启动器，自动填充选中文字并搜索。

**实现方案**：
- 监听系统剪贴板变化
- 需要剪贴板权限

**优先级**：⭐⭐

---

### 3. 使用统计

**描述**：记录每个应用的打开次数和最近打开时间，优先显示常用应用。

**实现方案**：
- 在 `canbox.json` 中添加 `launcherStats` 字段
- 记录每个应用的打开次数、最近打开时间
- 搜索时按匹配度 + 使用频率排序

**数据结构**：
```json
{
  "launcherStats": {
    "app-id-1": {
      "openCount": 10,
      "lastOpenTime": "2026-06-06T15:00:00Z"
    }
  }
}
```

**优先级**：⭐⭐⭐⭐

---

### 4. 动画效果

**描述**：窗口弹出/隐藏时添加动画效果（淡入淡出、滑动）。

**实现方案**：
- 使用 CSS `transition` 或 `animation`
- 弹出：从顶部滑入 + 淡入
- 隐藏：向顶部滑出 + 淡出

**优先级**：⭐⭐

---

## Phase 3：跨平台适配

### 1. Windows 系统应用读取

**描述**：读取 Windows 开始菜单的快捷方式。

**实现方案**：
- 读取路径：
  - `C:\ProgramData\Microsoft\Windows\Start Menu\Programs`
  - `%APPDATA%\Microsoft\Windows\Start Menu\Programs`
- 解析 `.lnk` 快捷方式（使用 `windows-shortcuts` 库）

**优先级**：⭐⭐⭐⭐

---

### 2. macOS 系统应用读取

**描述**：读取 macOS 的 Applications 文件夹。

**实现方案**：
- 读取路径：
  - `/Applications`
  - `~/Applications`
- 解析 `.app` 目录（实际上是一个目录，不是文件）

**优先级**：⭐⭐⭐

---

### 3. 平台特定的样式调整

**描述**：根据不同平台调整启动器窗口样式。

**实现方案**：
- Windows：使用系统主题色
- macOS：使用 Vibrancy 效果（毛玻璃）
- Linux：保持当前样式

**优先级**：⭐⭐

---

## 实施顺序建议

1. **Phase 2 - 使用统计**（最大价值，提升用户体验）
2. **Phase 3 - Windows 系统应用读取**（覆盖更多用户）
3. **Phase 2 - 计算器功能**（实用小功能）
4. **Phase 3 - macOS 系统应用读取**（覆盖 macOS 用户）
5. **Phase 2 - 网页搜索**（需要剪贴板权限，实现较复杂）
6. **Phase 2 - 动画效果**（体验优化）
7. **Phase 3 - 平台特定的样式调整**（精细化优化）
